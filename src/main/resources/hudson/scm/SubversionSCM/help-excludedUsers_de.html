Wenn Jenkins Subversion nach Änderungen befragt (um gegebenenfalls einen neuen Builds zu starten), werden alle Revisionen ignoriert, die von einem der hier angegebenen Committer eingecheckt wurden.

Bei diesem Ausschluß müssen Sie vollständige Subversion-Benutzernamen angeben. Bei mehreren Benutzernamen schreiben Sie diese in jeweils eine eigene Zeile. 

	 auto_build_user

Das obige Beispiel würde keine neuen Builds als Reaktion auf neue Revisionen des Committers "auto_build_user" auslösen.
