使用本參數後，建置時會要使用者選擇將從 Subversion 複製到工作區的 Tag。

一但名稱及儲存庫 URL 兩個欄位設定了，就必須:

  1. 確認作業使用 Subversion
  2. 將 Subversion 的儲存庫 URL 欄位設定成本參數裡兩個欄位值串接起來的值。
舉例來說，如果名稱是 SVN_TAG ，而儲存庫 URL 是 https://svn.jenkins-ci.org/tags，那麼 Subversion 的儲存庫 URL 就要設定成 https://svn.jenkins-ci.org/tags/$SVN_TAG。

注意，要是您把儲存庫 URL欄位設定成 Subversion 儲存庫的最上層，而不是指到 tags 目錄 (例如設定成 https://svn.jenkins-ci.org，而不是 https://svn.jenkins-ci.org/tags)。如果儲存庫根目錄裡有 trunk, branches 及 tags 三個目錄，那麼這個下拉選單就會讓使用者選擇要用 trunk 或是某一版的 branch 或 tag。
